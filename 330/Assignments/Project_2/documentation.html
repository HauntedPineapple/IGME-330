Noteworthy

The controls system is dynamic, the labels change when applicable to reflect the proper control usage. For example, the radio buttons that control the position are labelled "top", "middle", and "bottom" when the orietation is set to "horizontal", but when the orientation becomes "vertical", "top" becomes "left" and "bottom" becomes "right" and vice versa.

Grading

The project I created allows the user to mess around and create a pretty customized experience. Although even more controls could be added (and I plan to make this more robust in the future), I would say that this version has a sufficient enough amount for what was required in the assignment details. Overall I would say that this project gets an A from me.
